## 📄 About This Portfolio

The portfolio site is built with **vanilla HTML, CSS, and JavaScript — zero frameworks**. Inspired by the constraint that a strong developer doesn't need React to render a landing page.

```text
portfolio/
├── index.html      # Single-page layout with all sections
├── style.css       # Custom design system (CSS variables, no Tailwind)
├── script.js       # Typewriter, scroll progress, mobile nav, form handler
├── cheruu.png      # Logo / favicon source
├── CHRR.jpg        # Hero photo
├── codes.jpg       # Fallback hero image
└── Cherinet Asrat.pdf  # Downloadable resume
```

### Design highlights

- 🎨 Dark theme with electric blue (`#2979ff`) and cyan (`#00e5ff`) accents
- ✍️ Terminal-style animated role (`const role = "..."` typewriter)
- 🖱️ Desktop cursor glow + scroll progress bar
- 📱 Fully responsive, with a custom hamburger menu
- ⚡ Custom SVG iconography — no icon font dependency
